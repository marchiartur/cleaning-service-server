# Cleaning service project

## Description

Project to generate best route and manage users.

## version

node: 18.17.0
postgres: 14.5

## Installation

```bash
$ yarn
```

### Seed database

- create a database in postgres called `dev-cleaning-service`
- seed database

```bash
$ yarn seed
```

## Running the app (running default 3001)

```bash
# watch mode
$ yarn start:dev
```
